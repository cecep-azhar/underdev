# Under Construction Page

This is an Under Construction page built with Next.js 16.1.6.

## Features

- **Star Background**: Animated star field using HTML5 Canvas.
- **Countdown Timer**: A countdown to the launch date (customizable).
- **Theme Support**: Automatically adapts to system dark/light mode preference.
- **Modern Design**: Clean typography and responsive layout.
- **Vanilla CSS**: Global styles and CSS modules.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
